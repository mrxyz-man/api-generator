# vue-generator-api

Plugin for Vue 2, based on config file which create api extra axios instances by keyword with error-handler and set it into Vue proto.

## Project Setup

```sh
npm install
```

### Run demo for test

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
